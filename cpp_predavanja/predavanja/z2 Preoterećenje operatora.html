Preoterećenje operatora

Mladen Jurak
version 2.0, 2018-02

Zadatak: klasa Vector

Implementirati dinamički alocirani vektor s osnovnim operacijama koje dadu efikasno implementirati. Vektor implementira vrijednosnu semantiku (value semantics) što znači da pri kopiranju dolazi do kopiranja polja, a ne pokazivača. Sučelje je zadano na sljedećem slideu, a potrebno je implementirati sve metode i napraviti test program koji će verificirati ispravnost implementacije.

class Vector
{
    private:
        double *       m_data;    // Pokazivač na komponente
        unsigned int   m_size;    // Dimenzija vektora
    public:
        // Prazan vektor
        Vector() : m_data(0), m_size(0) {}
        // Vektor inicijaliziran nulama
        explicit Vector(unsigned int n, double val = 0.0);
        // CCtor
        Vector(const Vector& rhs);
        // Dtor
        ~Vector();
        // OP
        Vector & operator=(const Vector & rhs);
        Vector & operator=(double rhs);

        const double& operator()(unsigned int i) const {return m_data[i];}
        double& operator()(unsigned int i) {return m_data[i];}

        unsigned int size() const { return m_size; }
        // Promijeni veličinu vektora
        void resize(unsigned int new_size);

        // Složena pridruživanja koja se mogu efikasno implementirati
        Vector& operator+=(const Vector& rhs);
        Vector& operator-=(const Vector& rhs);
        Vector& operator*=(double scalar);
        Vector& operator/=(double scalar);

        // Jednakost vektora
        friend bool operator==(const Vector & lhs, const Vector & rhs);
};

Implementacija binarnih operatora

Implementacija operatora zbrajanja mogla bi biti ovakva:

Vector operator+(const Vector& lhs, const Vector& rhs)
{

    if(rhs.size() != lhs.size())
         throw std::invalid_argument("Vector dimensions mismatch!");

    Vector tmp(lhs.size(),0.0);

    for(unsigned int i=0; i < tmp.size(); ++i) tmp(i) = lhs(i) + rhs(i);
    return tmp;
}

Koliko je privremenih vektora konstrirano i destruirano u ovom kodu?

Vector a(1000000);
Vector b(1000000,0.1);
Vector c = b;

a = b + c;

Zadatak: Napravite test koji će usporediti vrijeme izvršavanja zbrajanja dva vektora pomoću operatora+() i vrijeme potrebno da se isti vektori zbroje po komponentama u jednoj for-petlji.

Efikasna implementacija binarnih operacija

Rješenje problema privremenih objekata: Operator zbrajanja ne smije vršiti zbrajanje već to treba prepustiti operatoru pridruživanja klase Vector. Operator zbrajanja treba samo formirati mali objekt koji sadrži sve informacije potrebne operatoru pridruživanja da izvrši operaciju nad vektorima.

U kodu a=b+c; operator pridruživanja (operator=()) je jedini koji može izvršiti traženu operaciju u vektoru a, bez kreiranja privremenih vektora.

Podaci potrebni operatoru pridruživanja:

  • Pokazivači na oba vektora

  • Duljine vektora

Operacije koje operator pridruživanja mora znati napraviti:

  • Povećanje pokazivača za jedan (prijelaz na sljedeći element u poljima b i c)

  • Operacija koju treba izvršiti.

Sve će te informacije biti zamotane u klasu čiji objekt vraća operator zbrajanja. Ostaje samo preopteretiti operator pridruživanja koji će uzeti taj objekt i izvršiti operaciju.

Operacije

Operacije pretvorimo u objekte tako da njima možemo parametrizirati druge klase

class Addition {
    public:
        static inline double apply(double a, double b)
        { return a+b; }
};

Zatim definiramo klasu BinExprOp<Op> koja sadrži informacije za operator pridruživanja. Ona je parametrizirana operacijom tako da za svaku novu operaciju ne moramo pisati novu klasu.

/ Klasa koja sadrži sve informacije potrebne za odgođeno
// izračunavanje binarne operacije.
template<typename Op>
class BinExprOp {
    private:
        const double * m_a; // pokazivač na početak prvog vektor
        const double * m_b; // pokazivač na početak druhog vektora
        unsigned int m_size; // duljina vektora - vektori moraju biti iste duljine
    public:
        // Konstruktor samo sprema podatke
        BinExprOp(const double * a,const double * b,unsigned int size):
        m_a(a),m_b(b),m_size(size){}
        // Konstruktor kopije i OP trebaju samo kopirati 3 podatka, tako
        // da su sintretizirani članovi dovoljni.
        // Za efikasnost je bitno da su sve tri sljedeće funkcije INLINE!
        unsigned int size() const { return m_size; }
        // Za iteriranje kroz oba polje treba nam operator ++
        BinExprOp& operator++() {++m_a; ++m_b; return *this;}
        // operator() vrši binarnu operaciju određenu parametrom Op na
        // Taj operator poziva operator pridruživanja iz klase Vector:
        //  Vector& Vector::operator=(BinExprOp<Op> result);
        double operator()() const { return Op::apply(*m_a, *m_b); }
//      ~BinExprOp() { std::cout << "~BinExprOp" << std::endl; }
};

Globalni operator zbrajanja:

BinExprOp<Addition>
operator+(const Vector& lhs, const Vector& rhs)
{
    assert(lhs.size() == rhs.size());
    return BinExprOp<Addition>(lhs.begin(), rhs.begin(), lhs.size());
}

Klasi Vector smo dužni dodati javnu funkciju begin() koja će vratiti pokazivač na prvi element vektora.

Operator pridruživanja koji dobije objekt tipa BinExprOp<Addition> lako će obaviti operaciju:

template<typename Op>
Vector& Vector::operator=(BinExprOp<Op> result)
{
    assert( m_size == result.size() );
    // Rezultat operacije smjesti u vektor
    for(unsigned int i=0; i < m_size; ++i){
        m_data[i] = result();   // Inline izraz
        ++result;
    }
    return *this;
}

Novi Vector

Nova verzija klase Vector mogla bi izgledati ovako:

class Vector
{
    private:
        double * m_data;         // Pokazivač na komponente
        unsigned int   m_size;    // Dimenzija vektora
    public:
        Vector() : m_data(0), m_size(0) {}
        explicit Vector(unsigned int n, double val = 0.0);
        Vector(const Vector& rhs);
        template<typename Op> Vector(BinExprOp<Op> rhs);
        ~Vector();

        Vector & operator=(const Vector & rhs);
        Vector & operator=(double rhs);
        template<typename Op> Vector& operator=(BinExprOp<Op> result);

        const double& operator()(unsigned int i) const {return m_data[i];}
        double& operator()(unsigned int i) {return m_data[i];}

        unsigned int size() const { return m_size; }
        void resize(unsigned int new_size);

        // Iteratori. Ovdje goli pokazivači.
        double * begin() { return m_data; }
        const double * begin() const { return m_data; }

        // Dade se efikasno implementirati
        Vector& operator+=(const Vector& rhs);
        //
        Vector& operator*(double x);
        friend bool operator==(const Vector & lhs, const Vector & rhs);
};

Napomena: Ubacili smo i konstruktor oblika Vector(BinExprOp<Op> rhs); kako bismo mogli pisati kod oblika:

Vector c = a + b;

Zadatak

Na opisani način implementirati globalne operatore zbrajanja, oduzimanja, množenja i dijeljenja (po komponentama) vektora. Testirati efikasnost operacija u odnosu na izvedbu s jednom for-petljom. Izmjeriti vrijeme izvršavanja u debug i optimiziranom modu.

Napomena: Ako imamo operatore množenja i dijeljenja skalarom u klasi Vector, onda možemo pisati efikasan kod oblika:

c = a*2 + b/3.0;

ali ne i kod oblika:

c = 2*a + b/3.0;
